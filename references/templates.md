# 文图反推提示词模板

Use these templates when the user asks to reverse text, images, or videos into prompts. Keep final answers concise.

## Spoken Idea To Prompt

Use this when the user speaks casually, gives a rough idea, or says "把我说的话变成提示词".

```text
我理解你的意思是：[用一句话整理用户真正想要的画面/视频/角色/场景]。

正向提示词：
请根据这段口语想法生成[图片/视频/角色/场景/海报]。核心主体是[主体]，处在[场景/环境]中，正在[动作/状态]。整体情绪是[情绪]，画面风格是[风格]，构图为[远景/中景/特写/俯视/低角度/正面等]，光线为[光线]，色彩以[主色]为主。需要把用户说的[关键意思]表达清楚，并补足必要的画面细节，但不要改变原本想法。

反向提示词：
偏离原意，过度改编，主体不清楚，场景错误，情绪错误，风格不一致，只堆关键词，细节无意义，文字，字幕，水印，logo，低质量，模糊，比例错误，肢体畸形

最需要保留：
[1-3个用户原话里的关键点]
```

## Text Reverse Prompt

```text
我理解这段文字的核心是：[主题 + 情绪 + 目标画面/输出]。

正向提示词：
请根据以下文字意图生成[图片/视频/场景/角色/海报]：核心主题是[主题]，主体是[人物/物体/场景]，处在[环境/背景]中，正在[动作/状态]。画面情绪是[情绪]，风格是[风格]，色彩以[主色]为主，光线为[光线]，构图为[构图/视角]。需要体现文字里的[关键含义/隐喻/设定]，不要只做表面元素堆叠。

反向提示词：
偏离文字主题，误解情绪，只堆砌关键词，主体不清楚，构图混乱，风格不一致，多余人物，文字，水印，logo，模糊，低质量，比例错误，肢体畸形

最需要保留：
[1-3 个关键词]
```

## Image Reverse Prompt

```text
我理解这张图的核心是：[主体 + 构图 + 氛围]。

正向提示词：
请生成一张[画面类型]。[主体]位于[构图位置]，[视角/景别]，[姿势/动作/表情]。[服装/道具/材质细节]清晰可见，背景是[环境/背景元素]。整体光线是[光线]，色彩以[主色]为主，搭配[辅助色]，画风为[插画/摄影/3D/手绘/像素/电影感等]，质感[精致/粗粝/柔和/闪粉/胶片/水彩等]。

反向提示词：
错误主体，错误姿势，错误构图，缺少关键细节，风格不一致，多余人物，文字，水印，logo，模糊，低质量，比例错误，肢体畸形，手部错误，过度写实，过度卡通

最需要保留：
[1-3 个关键词]
```

## Text + Image Fusion Prompt

```text
参考分工：
文字：作为[剧情/设定/情绪/主题]参考，保留[要保留的点]。
图片：作为[构图/姿势/风格/色彩/角色细节]参考，保留[要保留的点]。

正向提示词：
请以文字的[主题/设定/情绪]作为核心，以图片的[构图/姿势/风格/细节]作为视觉参考，生成一张[最终形式]。画面需要体现[文字关键含义]，同时保留图片里的[关键视觉元素]。整体风格是[风格]，色彩为[色彩]，光线为[光线]，构图为[构图]。

反向提示词：
文字含义丢失，图片风格丢失，参考优先级错误，主体不清楚，构图混乱，变成设定表，多视图，网格分栏，文字，标签，水印，logo，模糊，低质量
```

## Video Reverse Prompt

```text
我理解这段视频的核心是：[主体 + 动作 + 镜头 + 情绪]。

视频提示词：
生成一段[时长]视频。[主体]在[场景]中[动作]，镜头使用[固定/推镜/拉镜/跟拍/环绕/手持]，[景别/视角]，[运动节奏]。画面氛围是[情绪]，光线为[光线]，色彩为[色彩]，视觉风格是[风格/媒介]。保持[关键主体/动作/构图/关系]不变。

反向提示词：
动作错误，镜头乱晃，镜头过快，主体消失，主体变形，多余人物，错误场景，错误服装，文字，字幕，水印，logo，模糊，低质量，闪烁，断帧，肢体畸形，脸部崩坏
```

## Segmented Video Prompt

Use this when the user wants a video split into smaller generation prompts, such as every 10 seconds, every 15 seconds, or a 30-second clip.

```text
我理解这个视频需要分段生成：总时长[总秒数]秒，按[10/15/30]秒一段，共[段数]段。

统一风格锁定：
整支视频保持同一主体、同一角色外观、同一服装、同一场景空间、同一色彩倾向、同一画风和同一情绪基调。每一段都要能单独生成，但拼接后要像同一支连续视频。

分段提示词：
00:00-00:10：
[这一段的画面目标]。主体在[场景]中[动作/状态]，镜头[运动方式]，[景别/视角]，光线为[光线]，色彩为[色彩]，情绪为[情绪]。结尾保持[可衔接到下一段的状态]。

00:10-00:20：
延续上一段的角色、场景和光线。[这一段的新动作/变化]，镜头[运动方式]，节奏[快/慢/稳定]。结尾保持[下一段衔接点]。

00:20-00:30：
延续上一段。[这一段的动作推进/情绪变化/镜头变化]，保持同一视觉风格和主体一致性。

统一反向提示词：
角色变样，服装变样，场景突然变化，风格不一致，光线跳变，颜色跳变，镜头乱晃，动作断裂，主体消失，新增无关人物，脸部崩坏，肢体畸形，闪烁，断帧，文字，字幕，水印，logo，模糊，低质量
```

For 60 seconds every 10 seconds, use six sections:

```text
00:00-00:10：
00:10-00:20：
00:20-00:30：
00:30-00:40：
00:40-00:50：
00:50-01:00：
```

For 60 seconds every 15 seconds, use four sections:

```text
00:00-00:15：
00:15-00:30：
00:30-00:45：
00:45-01:00：
```

For 30 seconds, prefer:

```text
00:00-00:10：
00:10-00:20：
00:20-00:30：
```

## Seedance 2.0 Video Prompt

Use this when the user says the target model is Seedance 2.0. Keep prompts natural, cinematic, and segment-friendly. For long videos, split into clips of 10-15 seconds.

If the user mentions Doubao/Dreamina/Seedance fast mode, use the fast template below instead of a long normal prompt. Fast mode should be treated as preview/simple-shot mode: shorter clips, fewer actions, stronger reference lock.

```text
模型：Seedance 2.0

统一风格锁定：
[主体身份、角色外观、服装、场景、画风、色彩、光线、镜头质感]保持一致。整体是[风格]，画面[情绪]，[比例/画幅]。每段都可以单独生成，但拼接后要像同一支连续视频。

第1段：[时长]
[自然语言描述这一段：场景开场 + 主体状态 + 主要动作 + 镜头运动 + 光线色彩 + 结尾衔接状态]。

第2段：[时长]
延续上一段的角色外观、服装、场景和光线。[这一段的新动作/镜头变化/情绪推进]，结尾保持[下一段衔接状态]。

第3段：[时长]
延续同一角色和视觉风格。[这一段的动作推进或情绪变化]，镜头[运动方式]，保持画面稳定、流畅、可拼接。

反向提示词：
角色变样，服装变样，场景跳变，风格不一致，镜头剧烈乱晃，动作断裂，字幕，文字，水印，logo，脸部崩坏，肢体畸形，低质量，模糊
```

For a 38-second video, prefer four Seedance clips:

```text
第1段：0-10秒
第2段：10-20秒
第3段：20-30秒
第4段：30-38秒
```

For a 60-second video, prefer four 15-second clips or six 10-second clips:

```text
0-15秒 / 15-30秒 / 30-45秒 / 45-60秒
```

If action is fast, use 10-second clips. If action is slow and continuous, use 15-second clips.

## Seedance / Doubao Fast Mode Prompt

Use this when the user says fast version, quick version, Doubao fast, Dreamina fast, or when a long result drifts badly in later segments.

```text
模型：Seedance 2.0 / 豆包视频 Fast

模式判断：
Fast版用于预览和简单镜头，不适合一次生成复杂长视频。每段建议4-8秒，一个主要动作，一个镜头运动。正常/质量版可以写得更细，但长视频也要分段。

参考锁定：
整条视频始终保持同一个主体：[角色身份、发型、脸部特征、服装、道具]。始终保持同一场景：[地点、时间、天气、光线、色彩、画风]。不要新增无关人物，不要改变服装，不要改变角色年龄、脸型、发色、道具和主色调。

分段提示词：
第1段：0-6秒
[明确第一帧画面]。主体在[场景]中[一个动作]，镜头[一种运动方式]，光线[光线]，色彩[色彩]，风格[风格]。结尾停在[可作为下一段参考的画面]。

第2段：6-12秒
以上一段结尾画面作为参考帧。保持同一个主体、同一服装、同一场景、同一光线。[一个新的小动作]，镜头[一种运动方式]，结尾停在[下一段参考画面]。

第3段：12-18秒
以上一段结尾画面作为参考帧。继续保持参考锁定。[一个新的小动作或情绪变化]，动作不要突然跳跃，镜头稳定，结尾停在[下一段参考画面]。

统一反向提示词：
后半段漂移，角色变样，脸变样，发型改变，服装改变，道具消失，场景跳变，突然新增人物，动作过多，镜头乱晃，时间线崩坏，上一段衔接错误，风格不一致，颜色跳变，文字，字幕，水印，logo，脸部崩坏，肢体畸形，低质量，模糊
```

For a 38-second video in fast mode, prefer 6-7 short clips:

```text
0-6秒 / 6-12秒 / 12-18秒 / 18-24秒 / 24-30秒 / 30-36秒 / 36-38秒
```

For a 60-second video in fast mode, prefer 8-10 short clips:

```text
0-6秒 / 6-12秒 / 12-18秒 / 18-24秒 / 24-30秒 / 30-36秒 / 36-42秒 / 42-48秒 / 48-54秒 / 54-60秒
```

## Character Consistency / Continuity Prompt Pack

Use this when the user's main problem is character consistency, identity drift, outfit drift, or poor continuity between generated video segments.

```text
我理解这次的重点不是把提示词写长，而是保证人物一致和后续连续。

人物锁定：
同一个角色始终保持：[年龄感/脸型/眼睛/发型/发色/肤色/体型/服装/鞋子/道具/主色调/画风]。每一段都重复这些信息，不允许后续段自动发挥。

不可变信息：
- 脸部：[脸型、眼睛、表情气质、年龄感]
- 头发：[颜色、长度、发型、刘海、发饰]
- 服装：[上衣、裤子/裙子、鞋子、手套、头盔/帽子、关键图案]
- 道具：[必须保留的道具]
- 风格：[2D/3D/手绘/写实/卡通/电影感]
- 色彩：[主色、辅助色、光线]

可变信息：
- 表情可以轻微变化
- 姿势可以小幅变化
- 头发、衣服、粒子、雪雾、光影可以自然运动
- 镜头可以轻微推近、跟拍、平移，但不要大幅换角度

连续性方案：
不要一次生成完整长视频。把视频拆成短段生成。每一段的最后一帧要能作为下一段的参考帧。
第2段开始，使用上一段最后一帧作为参考/首帧，并在提示词里再次重复人物锁定。

分段提示词：
第1段：0-6秒
第一帧：[清楚描述角色正处于什么场景、什么姿势、穿什么、拿什么]。角色只做一个主要动作：[动作]。镜头：[一个简单镜头运动]。结尾画面：[适合作为下一段参考帧的稳定画面]。

第2段：6-12秒
使用上一段最后一帧作为参考帧。保持同一个角色、同一脸型、同一发型、同一服装、同一道具、同一场景和同一画风。角色继续做一个小动作：[动作]。结尾画面：[下一段参考帧]。

第3段：12-18秒
使用上一段最后一帧作为参考帧。再次保持人物锁定。角色继续：[动作]。镜头稳定，不突然换脸、不换衣服、不换场景。结尾画面：[下一段参考帧]。

统一反向提示词：
人物不一致，脸变样，年龄变化，发型变化，发色变化，服装变化，道具消失，体型变化，画风变化，后半段漂移，上一段和下一段接不上，时间线断裂，场景突然跳变，镜头角度变化过大，新增无关人物，动作过多，表情崩坏，肢体畸形，文字，字幕，水印，logo，低质量，模糊
```

For final-production prompts, prefer this output shape:

```text
版本：制作版
目标：人物一致 + 分段连续
模式：Fast / 正常 / 质量
建议分段：[每段秒数]
人物锁定：
连续性方案：
分段提示词：
统一反向提示词：
使用建议：
```

## Image-To-Video Prompt

```text
正向提示词：
请以上传图片作为首帧和主体外观参考，保持主体的长相、服装、配色、姿势关系、背景和整体氛围不变。只添加轻微自然运动：[头发/衣摆/翅膀/光点/烟雾/水面/镜头缓慢推进]。整体运动要克制、流畅、稳定，像原图自然活过来。

反向提示词：
改变角色长相，改变服装，改变构图，改变背景，动作过大，镜头剧烈晃动，主体消失，新增人物，画面闪烁，脸部崩坏，肢体变形，文字，水印，logo，模糊，低质量
```

## Style-Only Reverse Prompt

```text
正向提示词：
参考文字或图片的风格，不复制具体角色和构图。保留它的[线条/上色/材质/光线/色彩/笔触/颗粒/氛围/叙事语气]，用于生成一个新的[主体/场景]。

反向提示词：
直接复刻原图角色，复制原图构图，风格偏离，颜色错误，文字，水印，logo，低质量
```

## Anti-AI / IP Polish Prompt

Use this when the user says the image looks too AI-generated, too plastic, too generic, or they want a stronger original IP.

```text
我理解这次要解决的不是“细节不够”，而是让画面更像一个能长期使用的IP设定，并降低一眼AI感。

AI味诊断：
- [问题1：例如细节太平均、所有地方都在发光、画面没有主次]
- [问题2：例如角色像一次性生成物，没有稳定的识别符号]
- [问题3：例如材质太塑料、边缘太光滑、装饰太随机]

IP锁定：
- 角色轮廓：[最容易一眼记住的外形，例如短圆陶瓷人偶、破碎翅膀、低垂眼睛]
- 标志物：[固定道具或符号，例如燃烧心脏、飞蛾、裂纹、红色细线]
- 材质：[陶瓷/布料/金属/纸张/旧玻璃等，写清真实质感]
- 主色：[2-4个颜色，不要太多]
- 情绪：[痛苦、克制、重生、温柔、孤独等]
- 场景仪式：[这个角色经常出现的动作或环境]

人工修图方向：
以AI底图作为基础，进行人工二次修图和画面整理。减少无意义装饰，保留少量手工不完美边缘；强化主轮廓和主光源；让材质有真实磨损、裂痕、污渍和局部笔触；画面主次明确，背景不要抢角色。

正向提示词：
生成一张[画面类型/用途，例如IP主视觉、角色设定图、海报、图生视频首帧]。[主体]位于[场景]中，保持[IP锁定里的核心外形、标志物、材质、主色、情绪]。画面像AI初稿经过人工二次修图：构图被重新整理，细节密度有主次，边缘有少量手绘修正痕迹，材质逻辑真实，光影克制，颜色不脏不花。整体氛围是[情绪]，重点表达[主题]，不要只堆漂亮细节。

反向提示词：
一眼AI，AI生成感，塑料玩具感，过度光滑，过度精致，随机堆满装饰，所有区域细节平均，主体不清晰，主次混乱，完美对称，廉价梦幻滤镜，过饱和，过度锐化，假文字，乱码招牌，水印，logo，脸部崩坏，手部错误，肢体畸形，材质混乱，风格跳变，低质量，模糊

最需要保留：
[1-3个IP核心点]
```

For dark IP concepts, prefer this compact shape:

```text
正向提示词：
同一个[角色/陶瓷人偶/怪物/守护者]，保持[固定外形]、[固定材质]、[固定标志物]。画面主题是[痛苦到重生/守护/失忆/孤独]。构图简洁，有明确主光源和主轮廓，像AI底图经过人工二次修图，减少随机装饰，保留真实裂纹、烧痕、旧化、手工笔触和少量不完美。整体风格黑暗但不脏，情绪强，适合作为系列IP。

反向提示词：
一眼AI，塑料感，随机装饰堆满，廉价哥特，血腥内脏，恐怖猎奇，主体不稳定，角色变样，材质错误，过度发光，颜色太花，主次混乱，文字，水印，logo，低质量，模糊
```
